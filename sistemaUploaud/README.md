# Sistema Escolar - Upload de Arquivos

Sistema web desenvolvido para entrega de atividades escolares com upload, download, validação de arquivos e salvamento de metadados.

---

# Tecnologias utilizadas

- HTML
- CSS
- JavaScript
- Node.js
- Express
- Multer

---

# Funcionalidades

- Upload de arquivos
- Download de arquivos
- Exclusão de arquivos
- Validação de tamanho máximo
- Validação de MIME Type
- Salvamento de metadados
- Dashboard com estatísticas
- Interface administrativa escolar

---

# Tipos de arquivos permitidos

- PNG
- JPG
- PDF
